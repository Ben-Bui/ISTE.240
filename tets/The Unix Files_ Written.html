Linux Command Cheat Sheet

This page, created by Mary Brent, contains a list of linux commands with descriptions of thieir functions. The site is also organized by the type of command uses, such as basic commands, neetworking, or user management, in order to easily navigate the site to find what you need.

Linux Command Cheat Sheet 2

This page on GeeksForGeeks, is similar to the last, containging a list of linux commands broken into separate sections for easy navigation. This page also features a list of shortcut commands to help make using linux streamlined.

Chmod Overview

This page, located on linuxize.com, gives an overview of the chmod command, as well as a guide on how it should be used, including a list of character meanings to help in using the chmod command.

SED Command Guide

This page on GeeksForGeeks outlines the purpose of the SED command, which performs certain functions on files, and lists the uses of SED along with examples for each use.

AWK Command Guide

This page on GeeksForGeeks outlines the purpose of the AWK command, which is used for data manipulation. The page lists the AWK operations, its uses, and multiple examples for different uses.

Nslookup Command Guide

This page on GeeksForGeeks is an overview of the Nslookup command, which can retreive information from the DNS server. The page also contains a list of options for the command and some examples of its uses.
